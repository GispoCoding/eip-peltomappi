# **Peltomaahavainnon lisääminen**

1.  Paina alavalikon ***Lisää*** painikekeesta.

2.  Tarkista, että valittuna tasona on ***Peltomaahavainnot***.

3.  Siirry kartalla haluamaasi sijaintiin.

4.  Klikkaa alhaalta ***Tallenna***.

5.  Syötä halutut ominaisuustiedot joko +/- painikkeesta tai napauttamalla osiota ja syöttämällä arvon käsin.

6.  Valitse lopuksi oikean yläkulman vihreä ✅

Kohteelle tallennetaan tällä hetkellä seuraavat ominaisuustiedot:

-   Biologiset:
    <div>
    -   Lierot<br>
    -   Juuret<br>
    -   Juuret aktiivisuus<br>
    -   Nystyrät kpl/kasvi<br>
    -   Nystyrät aktiivisuus<br>
    -   Muu<br>
    </div>
-   Fysikaaliset:
    <div>
    -   Pintamaan vedenjohtavuus (aika, s)<br>
    -   Pintamaan vedenjohtavuus (syvyys, cm)<br>
    -   MARA - rakenneluokka<br>
    -   MARA - murut ja kokareet<br>
    -   MARA - huokoisuus<br>
    -   Murukestävyys (lusikkatesti)<br>
    </div>
-   Kemialliset:
    <div>
    -   ph<br>
    -   p<br>
    -   k<br>
    -   ca<br>
    -   mg<br>
    </div>
 
Uuden kohteen lisääminen sijoittuu automaattisesti omaan sijaintiisi. Tämän voi muuttaa asetuksista, napauta oikeasta alakulmasta ***Lisää -> Asetukset -> Lukitse automaattisesti sijainti***. Kun asetus on pois päältä, uusi kohde sijoittuu aina karttanäyttösi keskikohtaan.

Mikäli ominaisuustiedoista puuttuu jokin tai haluat muutoin antaa palautetta ominaisuudesta, käytä alla olevaa lomaketta:
